# 📈 Rate Shock Explorer

> How Federal Reserve rate hikes ripple through global markets — an interactive event study.

## Overview

This project analyses the short-term market impact of every Federal Reserve rate hike from 2015 to 2023, across five global assets:

- **SPY** — US Stock Market (S&P 500 ETF)
- **GLD** — Gold
- **DXY** — US Dollar Index
- **CNY/USD** — Chinese Yuan / US Dollar exchange rate
- **DGS10** — 10-Year US Treasury Yield

Using event window analysis, the project tracks cumulative asset returns from 5 trading days before to 30 days after each hike, and compares how different hike sizes and cycles produce different market reactions.

## How to Run

### Option 1: View the live app (recommended)

👉 **[Open Rate Shock Explorer on Streamlit Cloud]([https://ezakzrxxmwndyrxinghthe.streamlit.app])**

### Option 2: Run locally

**1. Clone the repository**
```bash
git clone https://github.com/huhng24/rate-shock-explorer.git
cd rate-shock-explorer
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Run the Streamlit app**
```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`.

**4. (Optional) Re-run the analysis notebook**

Open `explore.ipynb` in Jupyter to see the full data cleaning and analysis workflow.

## App Features

| Page | Description |
|---|---|
| 📉 Event Windows | All hike events overlaid — see each asset's reaction pattern |
| 📊 Average Impact | Average return by asset across different time horizons |
| 🔎 Hike Size vs Reaction | Compare how 25bp / 50bp / 75bp hikes affect each asset |
| 🗓️ Single Hike Detail | Deep-dive into any individual hike with market context |
| 🗃️ Data Table | Browse and download the cleaned dataset |

**Interactive controls (sidebar):**
- Select assets to display
- Filter by rate hike cycle (2015–2018 / 2022–2023 / All)
- Adjust observation window (days before and after hike)
- Select analysis horizon (Day +1 to Day +30)
- Reset to default settings

---

## Key Findings

- **DGS10** responds most consistently — yields rise reliably after every hike
- **DXY** strengthens after large hikes (75bp), reflecting capital inflows to USD assets
- **SPY** shows the highest dispersion — equity markets respond to *surprise* more than hike size
- **GLD** tends to decline in the weeks following hikes as real rates rise
- The **2022–2023 cycle** produced significantly larger reactions than **2015–2018** across all assets
